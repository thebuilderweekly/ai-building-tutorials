# Ship an llms.txt that tells agents what your site is

An agent that reads a well-structured llms.txt picks up your entire corpus in one pass, which a human writing documentation by hand would take months to make this legible.

This is part of [AI Building Tutorials](https://github.com/thebuilderweekly/ai-building-tutorials) by [The Builder Weekly](https://thebuilderweekly.com).

**Read this tutorial:**
- [In this repo](./tutorial.md) — the raw markdown with code blocks
- [On the web](https://thebuilderweekly.com/tutorials/llms-txt-as-contract) — rendered with diagrams and syntax highlighting

## What this tutorial teaches

**Before:** An AI crawler visiting your site sees hundreds of pages with no hint about which ones matter or how they relate.

**After:** The crawler reads llms.txt, gets a guided tour of your corpus, and indexes the pages that matter in the order you chose.
