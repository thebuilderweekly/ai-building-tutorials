# Communication

Patterns for agents that interact with humans through messages. Email, chat, notifications. Tutorials in this pillar teach reading inboxes, drafting responses, and gating sends through human approval so the agent never sends something embarrassing.

## The pattern

Every communication tutorial follows the same arc: an agent reads or writes human-facing messages, the risk is sending something wrong or missing something important, and the tutorial shows how to gate the high-risk actions through verification or approval.

- **Before-state:** "You manually process hundreds of messages a day with declining accuracy."
- **After-state:** "The agent handles classification and drafting; you approve the outputs that matter."
